# 📖 Project Overview

Credit card fraud has become one of the most significant challenges in the financial industry. Millions of digital transactions are processed every day, making it difficult for financial institutions to identify fraudulent activities manually.

This project presents a Machine Learning-based Credit Card Fraud Detection System that analyzes transaction-related information to classify transactions as **Fraudulent** or **Legitimate**.

The project includes:

- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Fraud Prediction System
- Interactive Dashboard
- Website Integration

---

# 📂 Dataset Information

The dataset contains transaction-level information related to credit card activities.

### Dataset Features

| Feature | Description |
|----------|-------------|
| Transaction ID | Unique transaction identifier |
| Customer Name | Customer information |
| Merchant Name | Merchant details |
| Transaction Date | Date of transaction |
| Transaction Amount (INR) | Amount spent |
| Fraud Risk | Risk level of transaction |
| Fraud Type | Type of fraud |
| State | Customer state |
| Card Type | Credit/Debit card type |
| Bank | Bank name |
| Fraud Score | Fraud probability score |
| Transaction Category | Shopping, Food, Travel, etc. |
| Merchant Location | Merchant location |
| IsFraud | Target Variable |

# 🏆 Best Model

After evaluating multiple algorithms,

**Random Forest** achieved the best overall performance.

### Performance

- Accuracy : **72.5%**
- Precision : **66.67%**
- Recall : **7.02%**
- F1 Score : **12.70%**

Random Forest was selected as the final prediction model because it provided the highest accuracy among the tested algorithms.
